🚀 DEMARRAGE EN 3 ÉTAPES:

ÉTAPE 1️⃣ - Lancer Ollama (Terminal 1)
  $ ollama serve
  ↳ Laissez ce terminal ouvert

ÉTAPE 2️⃣ - Télécharger un modèle (Terminal 2)
  $ ollama pull mistral
  ↳ Télécharge Mistral (4-5GB) - Meilleur pour RTX 3070 8GB

ÉTAPE 3️⃣ - Lancer Jarvis
  • Double-cliquez sur jarvis.bat
  OU
  $ python main.py

❓ TROUBLESHOOTING:

❌ "Ollama connection refused"
   → Vérifiez que ollama serve est lancé

❌ "Model not found"
   → ollama pull mistral

❌ "Out of Memory"
   → Utilisez un modèle plus léger (orca-mini)

❌ "Python command not found"
   → Utilisez le chemin complet dans jarvis.bat
